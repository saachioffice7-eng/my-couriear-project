# ParcelFlow - Quick Start Guide

## ✅ Authentication Fixed!

The OTP issue has been resolved. The system now uses **Email + Password** authentication, which is more reliable and works immediately without waiting for OTP codes.

---

## 🚀 How to Get Started

### 1. Register Your Account

1. Open the application
2. Click **"Register"** on the login page
3. Fill in the form:
   - Full Name
   - Business Name
   - Email
   - Phone (optional)
   - Password (at least 6 characters)
   - Confirm Password
4. Click **"Create Account"**
5. ✅ You're automatically logged in!

### 2. Login (Next Time)

1. Enter your **Email**
2. Enter your **Password**
3. Click **"Sign In"**
4. ✅ You're in!

---

## 📋 What to Do After Login

### Step 1: Add Courier Companies

Go to **Courier Companies** page and add your courier partners:
- Company Name (e.g., "Delhivery", "Blue Dart")
- Contact Person
- Phone
- Office Address

### Step 2: Set Up Rates

Go to **Rates** page and configure pricing:
- Select Courier Company
- Enter City name
- Set weight range (min-max in grams)
- Set price for that range

**Example:**
- Delhivery → Mumbai → 0-1000g → ₹25
- Delhivery → Mumbai → 1000-2000g → ₹50

### Step 3: Start Adding Parcels

Go to **Parcels** page:
- Click "Add Parcel"
- Fill in customer details
- Select courier company and city
- Enter weight
- **Price auto-calculates!** ✨
- Save and print label

## 🔒 Security

- All data is protected by Row Level Security (RLS)
- First user becomes Admin automatically
- Each user can only access their own data
- Secure password-based authentication

## 💡 Pro Tips

1. **Set up rates first** - This enables auto price calculation
2. **Use consistent city names** - Makes filtering easier
3. **Add expense categories** - Better expense organization
4. **Export reports regularly** - Keep backups of your data

## 🆘 Need Help?

### Can't Login?
- Make sure email and password are correct
- Password is case-sensitive
- Try registering a new account if needed

### Data Not Saving?
- Make sure you're logged in
- Check your internet connection
- Refresh the page and try again

### Price Not Auto-Calculating?
- Make sure you've added rates for that courier + city combination
- Check that weight is entered correctly
- Verify courier company and city are selected
